This is a simple carousel slider with light box effect, It\'s shows your selected image as a carousel.

== Description ==
Light Box is a fully responsive carousel gallery plugin for any site. Gust create a slide with feature image on it, you can add as many slide as you want. You can publish it with single shortcode [lightbakso_carousel]. This carousel plugin has various settings like auto play, loop, number of items,navigation buttons and much more. 

**How to use [lightbakso_carousel] shortcode?**

Copy and paste [lightbakso_carousel] shortcode where you want to show the carousel. you can publish this carousel on any Page or Post on your WordPress website.

**Features:**

* Full Responsive
* Can choose number of items show in different device.
* Center the items.
* Two type of navigation
* Auto Play and auto play speed
* Custom style
* Advance customization in WordPress customize panel.   
* Full width carousel.

== Installation ==

1. Upload the entire lightbakso folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress admin.
3. Add as many new slide as you want.
4. Use [lightbakso_carousel] shortcode to publish your carousel into any page or post.
